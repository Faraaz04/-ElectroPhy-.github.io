DAC with R-2R Ladder

This project involves building a Digital-to-Analog Converter (DAC) using an R-2R ladder network. The R-2R ladder is a resistor network that converts digital input signals into analog output signals.

WHAT IS R-2R?

It means we have used built a ladder network using 2 sets of resistors of which the second has x2 value of resistance than the previous.

Advantage: Only resistors of 2 different values are required.
